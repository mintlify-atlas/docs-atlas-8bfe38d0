## How it works

The system processes video through a three-stage pipeline — temporal frame sampling, YOLOv8 human masking, and TSN classification — returning a labeled prediction with confidence score and visual explanations.

<CardGroup cols={3}>
  <Card title="TSN Model" icon="brain" href="/concepts/tsn-model">
    ResNet-18 backbone trained on 8-segment temporal sampling for video-level classification
  </Card>
  <Card title="Human Masking" icon="person" href="/concepts/human-masking">
    YOLOv8 detects persons and dims background regions to reduce noise
  </Card>
  <Card title="Grad-CAM" icon="eye" href="/concepts/gradcam">
    Heatmap overlay showing which spatial regions drove the prediction
  </Card>
</CardGroup>

## Supported video formats

The API accepts **MP4, AVI, MOV, MKV, WebM, and M4V** video files via multipart form upload.

## Web client

A standalone browser client (`client/index.html`) provides drag-and-drop upload and live 8-second camera recording — no additional tooling required.
